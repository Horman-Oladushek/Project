Проект Python, PyQt

Название: магазин электроники на PyQt

Цель: изучить библиотеку Qt, на языке программирования Python

Возможности: входить в систему под обычным пользователем или админом, искать товары в базе данных,
добавлять и удалять товары  (только для админов), так же присутствует возможность регистрации.
Из всех заявленных функций на момент заявления темы, не хватает только добавление товаров в корзину и привязки
их к обычным пользователям

Описание:

ГЛАВНОЕ ОКНО:

Поиск - работает по принципу всех поисковых строк) То есть, вам нужно вбить интересующий товар (или первые несколько
букв), после чего, нажать на кнопку "Найти". В таблице с базой данных выделятся товары, подходящие к вашему запросу

Вход в систему осуществляется при нажатии кнопки Войти в правом верхнем углу экрана.
При нажатии на эту кнопку, появляется окно входа, в котором можно войти как обычный пользователь,
так как и администратор, а так же зарегестрироваться под видом обычного пользователя.

Если вы зашли под аккаунтом админа, вам предоставляется право добавить новый товар, нажав кнопку "Добавить", после
в новом окне, вы добавляете информацию о товаре. Так же, есть возможность удалить его, нажмите соответствующую кнопку
"Удалить"

После закрытия окна входа или окна удаление/добавления товаров, нужно нажать кнопку "обновить", для
обновления данных о пользователе и базы данных товаров

ОКНО ВХОДА В СИСТЕМУ:

Для входа в администратора нужно ввести логин и пароль из базы данных "Base_adm.csv",
анологично для пользователей - из "Base_users.csv".

Нужно ввести логин и пароль и нажать соответсвущую кнопку для входа:

"Войти как админитратор" - вход по данным админинстратора
(Login: admin, Password: admin)

"Войти" - вход по данным пользователей
(      login:           Password:
       user             проверка
       user_2           проверка_2
       Horman           oladushek)


"Регистрация" - регистрация нового пользователя и добавление его в базу, после регистрации нажмите "Войти"


ОКНО УДАЛЕНИЯ/ДОБАВЛЕНИЯ:

При выборе одного из этих окон, появляется либо окно с удалением товара, либо с его добавлением

При удалении нужно ввести полное название товара (идея взята с проверки удаление репозиториев на гитхабе), после
нажать кнопку удалить. Рекомендую заранее копировать название товара

При добавлении товара, вам нужно заполнить в определенных строках информацию о нем: Название, Цена, Количество,
Места самовывоза (только под конец вспомнил про это название, во всем проекте называется "Где можно забрать")
