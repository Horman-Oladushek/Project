Проект, PyQt

Возможности: входить в систему, искать товары в базе данных, добавлять и удалять товары  (только для админов),
так же присутствует проверка пароля и логина при входе

вход осуществляется при нажатии кнопки Войти в правом верхнем углу экрана

при нажатии на эту кнопку, появляется окно входа, в котором можно войти как обычный пользователь,
так как и администратор.

Для входа в администратора нужно ввести логин из базы данных "Base_adm.csv", для пользователей
из "Base_users.csv"

Нужно ввести логин и пароль и нажать соответсвущую кнопку для входа:
"Войти как админитратор" - вход по данным админа
"Войти" - вход по данным пользователей

После закрытия окна входа, нужно нажать кнопку "обновить", для обновления данных

если вы зашли под аккаунтом админа, вы можете добавить новый товар, нажав кнопку "Добавить", после
в новом окне, вы добавляете информацию о товаре
