Проект Python, PyQt, SQL.

Название: автоматизация процесса обслуживания покупателей

Цели в файле "Проект 2023-24"
так же вся теория в нем
тут же представленна практика, которая будет дорабатываться и доделываться, убираться костыли, палки и делать все по человечески


Основные задачи, что нужно поменять:
перейти с csv на sql
переписать функции с таблицей товаров, проверки входа, добавление/удаление товаров





Возможности:
регистрация в системе под новой учетной записью;
авторизация в системе под рабочей учётной записью и учётной записью администратора системы;
добавлять/удалять товары  (функционал уч. записи администратора);
поиск товара пользователем (Функция добавление товаров в корзину и привязки к пользователям - на текущий момент недоступна).

Описание:

ГЛАВНОЕ ОКНО:

Вход в систему осуществляется при нажатии кнопки "Войти" в правом верхнем углу экрана.
При нажатии на эту кнопку, появляется окно входа, в котором необходимо ввести учетную запись. Система позволяет зарегестрировать 
новую учетную запись. Используются 2 типа учетных записей: пользовательская учетная запись и учетная запись администратора. 

Учетная запись администратора позволяет добавить новые товары (нажав кнопку "Добавить", после в новом окне, вы добавляете информацию о товаре).
Так же, есть возможность удалить товар, нажмите кнопку "Удалить".

После закрытия окна входа или окна удаление/добавления товаров, нужно нажать кнопку "Обновить", для
обновления данных о пользователе и базы данных товаров.

Поиск - Необходимо ввести наименование товара (или первые несколько букв), после чего, нажать на кнопку "Найти".
В таблице с базой данных выделяются товары, подходящие к вашему запросу.

ОКНО ВХОДА В СИСТЕМУ:

Для входа в учетную запись администратора нужно ввести логин и пароль из базы данных "Base_adm.csv".
Для входа в учетную запись пользователя нужно ввести логин и пароль из базы данных "Base_users.csv".

пример:
"Войти как админитратор" - вход по данным админинстратора.
(Login: admin, Password: admin)

"Войти" - вход по данным пользователей.
(      login:           Password:
       user             проверка
       user_2           проверка_2
       Horman           oladushek)


"Регистрация" - регистрация нового пользователя и добавление его в базу, после регистрации нажмите "Войти".


ОКНО УДАЛЕНИЯ/ДОБАВЛЕНИЯ:

При выборе одного из этих окон, появляется окно с удалением/добавлением товара.

При удалении нужно ввести полное название товара (идея взята с проверки удаление репозиториев на гитхабе), после
нажать кнопку "Удалить". Рекомендуется заранее копировать название товара, для исключения ошибок удаления.

При добавлении товара, необходимо заполнить соответствующие поля: Название, Цена, Количество, Где можно забрать.
